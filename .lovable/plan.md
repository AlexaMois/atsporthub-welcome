# Уникальные иконки для каждого пункта боковой навигации

## Проблема

Сейчас пункты меню внутри групп (ВЧНГ, СН, ГПНЗ и т.д.) отображаются без иконок -- только текст и счётчик. Это затрудняет визуальную навигацию.

## Подход

Поскольку названия пунктов загружаются динамически из внешнего API (их набор может меняться), невозможно вручную привязать иконку к каждому элементу. Решение:

1. **Создать маппинг известных названий на иконки** -- для часто встречающихся пунктов (ВЧНГ, СН, ГПНЗ, Водитель, Механик и т.д.) зашить соответствие "название -> иконка Lucide".
2. **Для неизвестных пунктов** -- назначать иконку автоматически из пула иконок по индексу (циклически), чтобы каждый пункт получил визуально уникальную иконку.
3. **Иконки отрисовываются слева от текста** в каждом `SidebarMenuButton`, размер h-3.5 w-3.5, цвет наследуется от текста.
