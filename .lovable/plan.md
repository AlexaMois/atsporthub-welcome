

# Корпоративный портал документов АТС — Экран выбора роли

## Дизайн-система
- Primary: `#0099ff`, фон: `#f5f7fa`, карточки: `#ffffff`, текст: `#0a1628`
- Шрифт: Inter (Google Fonts)
- Обновление CSS-переменных в `index.css` под указанные цвета

## Экран: Выбор роли (`/`)
- **Шапка** — фиксированная, фон `#0099ff`, белый текст «АТС Портал» слева, высота 56px
- **Заголовок** — «Кто вы?», жирный, text-2xl, отступ сверху
- **Сетка карточек** — 2 колонки на мобильном, 3 на desktop
  - Белая карточка, rounded-xl, shadow-sm, padding, hover-эффект
  - Иконка Lucide сверху (Car, Wrench, Radio, Package, UserCheck, Crown)
  - Название роли под иконкой
  - У «Генеральный директор» — иконка замка Lock в правом верхнем углу цвета `#0099ff`
- **Клик** — console.log с именем роли, навигация на `/role/:roleName`
- Страница `/role/:roleName` — заглушка «Раздел для [роль]»

## Supabase
- Подключение Supabase-клиента (инициализация), без использования на данном этапе

## Данные
- Статический массив ролей, без бэкенда
